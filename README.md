# Multi-Agent-Cooking-Assistant

## Overview
Multi-agent cooking assistant built with FastAPI, Pydantic, LangChain, and asyncio. It understands food-related queries, identifies intent, routes them to a chef or grocer agent, and returns structured responses with a clean, modular backend design.
